Maven Recipe #0

May 16th, 2010

问题： 多个root pom的dependencyManagement有重复的内容，希望统一管理。
解决：
新建一个空pom.xml，在dependencyManagement中指定这些依赖，如
[cc lang=”xml”] xsi:schemaLocation=”http://maven.apache.org/POM/4.0.0 http://maven.apache.org/maven-v4_0_0.xsd”>

4.0.0 info.sunng root pom 0.0.1-SNAPSHOT

info.sunng X 0.0.1-SNAPSHOT

[/cc]
将这个pom.xml部署到你的Maven私服。

在其他root pom中添加这样的dependencyManagement
[cc lang=”xml”]
….
info.sunng root 0.0.1-SNAPSHOT pom import ….
[/cc]
