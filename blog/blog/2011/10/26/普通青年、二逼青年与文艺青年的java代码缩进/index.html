Here comes the Sun

普通青年、二逼青年与文艺青年的Java代码缩进

Oct 26, 2011

普通青年

[cc lang="java"]
while(true) {
if (something) {
System.out.println(something);
break;
}
} [/cc]
特点： tab与空格混用，无其他特点。
常见于：各类代码仓库。

二逼青年

[cc lang="java"]
while(true)
{

if (something)
{
System.out.println(something);
break;
}

}
[/cc]
特点：总担心代码不够长
见于：各类劣质技术书籍

文艺青年
[cc lang="java"]
while(true)
{
if (something)
{
System.out.println(something);
break;
}
}
[/cc]
特点：普通Java青年永远不会理解的缩进，lisp程序员会心一笑
见于：Clojure源码
