Exaile DoubanFM Plugin 0.0.9

新版本的Exaile豆瓣电台插件。这个版本的变更：

  • 新增分享菜单，分享当前音轨到豆瓣、新浪、开心001、人人
  • 改进了推荐方式，采用豆瓣电台的永久链接
  • 修改了0.0.8以来存在的登录bug
  • 修改了0.0.8以来存在的加载播放列表bug

从github获得新版本： https://github.com/sunng87/exaile-doubanfm-plugin/downloads
