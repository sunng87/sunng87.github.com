Python获取本机IP

除了用inetface库和分析ifconfig输出以外，一个比较简单的获取本机IP的方法：
[cc lang="python"]
import socket
def get_local_ip():
s = socket.socket(socket.AF_INET, socket.SOCK_DGRAM)
s.connect(("sdo.com",80))
return s.getsockname()[0]
[/cc]
