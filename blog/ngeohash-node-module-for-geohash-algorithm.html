ngeohash是一个geohash的javascript实现，之所以叫做ngeohash是因为到了Publish的时候才发现已经有geohash这个module了。这么令人沮丧的事就不多说了。

安装 npm install ngeohash

使用
var geohash = require('ngeohash');
sys.puts(geohash.encode(32.1717, 118.2342));
