YAMB(Yet Another Microblogging Tool) which is mentioned several days ago is now hosted on Sun's project-kenai licensed with GPL2. http://kenai.com/projects/yamb

You can grab the source code from following link via mercurial(hg). https://kenai.com/hg/yamb~hg-repos

Actually, there is nothing new.

Proudly powered by Pelican, which takes great advantage of Python.
